# Student Management System (Java Swing)
A desktop-based Student Management System developed using Java Swing, OOP concepts, and ArrayList.

## Features

* Add Student
* View All Students
* Search Student by Roll Number
* Update Student Information
* Delete Student
* Clear Form
* Input Validation
* Duplicate Roll Number Check
* Confirmation Dialog Before Deletion

## Technologies Used

* Java
* Java Swing
* OOP (Object-Oriented Programming)
* ArrayList
* Exception Handling
* Event Handling

## Validation Implemented

* Empty Fields Validation
* Numeric Roll Number Validation
* Duplicate Roll Number Validation
* Student Existence Check

## Project Structure

* Student Class
* GUI Components (Swing)
* CRUD Operations
* Validation Logic
